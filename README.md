# Audiosplit

## Description

## Usage

```bash
$ audiosplit -h
usage: cut.py [-h] [-i INPUT] [-I INPUT_TRACKLIST] [-t TRACKLIST] [-s] [-o OUTPUT_DIR]

Cut MP3 file into parts based on tracklist.

options:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Input MP3 file
  -I INPUT_TRACKLIST, --input-tracklist INPUT_TRACKLIST
                        Input tracklist file
  -t TRACKLIST, --tracklist TRACKLIST
                        Tracklist as a string
  -s, --skip            Skip the confirmation step
  -o OUTPUT_DIR, --output-dir OUTPUT_DIR
                        Output directory

Example: cut.py -i input.mp3 -t "00:00:00 Track 1
00:01:00 Track 2"
```
